Complete examples of using calendars as input field datepickers

1. Basic DatePicker

2. Ethiopian Calendar DatePicker

<input type="text" id="ethiopian-datepicker"> <script> new ModernDatePicker('#ethiopian-datepicker', { calendar: 'ethiopian', language: 'am', theme: 'green' }); </script>
Selected: None

3. Auto-Initialize with Data Attributes

<input type="text" data-datepicker='{"calendar":"gregorian","theme":"purple","format":"dd/mm/yyyy"}' placeholder="Auto-initialized datepicker">
